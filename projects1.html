Project 1: E-commerce Website Testing

Details about the E-commerce Website Testing project.

Overview

This project involved performing end-to-end manual testing for an e-commerce platform. The goal was to ensure a seamless user experience, identify bugs, and validate the functionality of key features such as product search, checkout, and payment processing.

  • Performed functional, regression, and exploratory testing.
  • Collaborated with developers to resolve identified issues.
  • Validated payment gateway integration and ensured secure transactions.

Tools Used

JIRA for defect tracking, TestRail for test case management, and browser developer tools for debugging.

Outcome

The project was successfully delivered with zero critical defects, resulting in a smooth launch and positive user feedback.
