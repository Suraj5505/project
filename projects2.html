Project 2: Healthcare Application

Details about the Healthcare Application project.

Overview

This project involved testing a healthcare application for managing patient records. The focus was on ensuring data accuracy, security, and compliance with healthcare regulations.

  • Validated patient record management features.
  • Ensured compliance with HIPAA and other healthcare standards.
  • Performed security testing to protect sensitive patient data.

Tools Used

JIRA for defect tracking, Postman for API testing, and SQL for database validation.

Outcome

The project was delivered successfully, ensuring a secure and reliable application for healthcare providers.
